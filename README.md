# Shreya Katoch Portfolio 🧬

This is a premium, research-focused computational biology portfolio.

### 🛠️ How I Made It
- **Design**: Built with a **Glassmorphism** aesthetic using modern CSS3 (Blur, Gradients, Grid).
- **Tech**: 100% Vanilla **HTML5, CSS3, and JavaScript** (No heavy frameworks, super fast).
- **Interactivity**: 
  - **3D Tilt**: Perspective-based mouse tracking on cards.
  - **DNA Particles**: Custom Canvas API animation.
  - **Dual Mode**: Built-in Light/Dark theme engine.

### 🚀 How to Open
- **Live Version**: Go to **[shreyakat0ch.github.io](https://shreyakat0ch.github.io)**
- **Local Version**: Just open the `index.html` file in any web browser (Chrome, Safari, etc.).
