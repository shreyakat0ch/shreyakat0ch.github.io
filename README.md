# Shreya Katoch — Computational Portfolio 🧬

A premium, world-class computational biology portfolio featuring advanced glassmorphism aesthetics, interactive 3D elements, and research-focused storytelling.

**Live Site:** [shreyakat0ch.github.io](https://shreyakat0ch.github.io)

## 🌟 Key Features
- **Master's Profile Integration**: Custom sections for M.Sc. Data Science & AI and B.Sc. Physical Sciences.
- **Glassmorphism Design**: High-end translucent UI with soft "light-brown/cream" aesthetics.
- **Interactive 3D Elements**: Magnetic 3D tilt effects on project cards and profile avatar.
- **DNA Particle Background**: Custom DNA-inspired canvas animation with dynamic connections.
- **Dual Theme Support**: Seamless transition between "Latte Light" and "Espresso Dark" modes.
- **Research Journey**: Interactive timeline showcasing internships (CSIR-IHBT) and flagship projects.

## 🛠️ Tech Stack
- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom Properties, Flexbox/Grid, Backdrop Filters)
- **Logic**: Vanilla JavaScript (ES6+)
- **Interactivity**: Intersection Observer API, Canvas API, 3D Perspective Transforms

## 📂 Project Highlights
- **Histone Mark Predictor**: CNN-Transformer architecture for epigenomic prediction.
- **Causal GRN Pipeline**: 5-stage causal inference for 14K gene networks.
- **Deep Learning in Bioinformatics**: Research internships and technical contributions.

## 🚀 Local Development
To run the portfolio locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/shreyakat0ch/shreyakat0ch.github.io.git
   ```
2. Open `index.html` in any modern browser.
3. (Optional) Use a local server like `Live Server` in VS Code for best experience.
